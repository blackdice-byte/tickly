Tickly is a beautiful and intuitive todo app to help you get things done.

### Features

**Task Management**
- Create, complete, and delete tasks
- Set priority levels (None, Low, Medium, High) with color-coded indicators
- Add subtasks to break down complex tasks
- Track subtask progress

**Organization**
- Create custom projects to organize tasks
- Default Inbox for quick task capture
- Color-coded projects for easy identification
- Sort tasks by date, priority, or alphabetically

**Tags & Labels**
- Tag tasks with customizable labels
- Pre-built tags: Work, Personal, Urgent
- Filter and organize with visual tag badges

**Customization**
- Light and Dark theme support
- System theme auto-detection
- Settings accessible from main screen

**User Experience**
- Animated splash screen with smooth transitions
- Expandable task cards for detailed editing
- Haptic feedback on interactions
- Clean, modern interface

### Technical Details
- Built with Expo SDK 54
- React Native 0.81.5
- Supports iOS and Android
